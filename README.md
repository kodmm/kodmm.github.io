# kodmm.github.io

個人のキャリア・プロフィール用サイト（GitHub Pages）。

## 公開までの手順

1. リポジトリ **Settings** → **Pages**
2. **Build and deployment** → Source: **Deploy from a branch**
3. Branch: **main** / Folder: **/ (root)**
4. 数分後に **https://kodmm.github.io/** で表示されます。

## 構成

`index.html` にトップと各章（アンカー）を集約。社名・固有名は載せていません。
